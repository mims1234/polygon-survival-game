# Polygon Survival Game (Wildlands)

A grid-based, top-down polygon survival and base-building game. Explore the region, gather tools and resources, and advance your empire from a campfire to a thriving settlement!

## Features
- **Exploration & Regions**: Discover procedurally generated biomes through a fog-of-war grid. Walk to the map boundaries to travel across expanding world coordinates.
- **Resource Gathering & Tools**: Harvest wood, stone, food, and water. Upgrade your tools (Axe, Pickaxe, Bucket) at the Forge to increase yield efficiency.
- **Base Building & Crafting**: Craft necessities ranging from Campfires for warmth to Markets for trading. Utilize Automation systems via your structured economy.
- **Survival Mechanics**: Manage stamina and hunger, endure dynamic day/night cycles, and earn progression XP as you play. 

## Technology Stack
Built using Vanilla HTML/CSS/JavaScript and leveraging the **[Three.js](https://threejs.org/)** library to render the world layout, player animations, and game environment. 

## Getting Started
To test and run the project locally, any standard local web server will work:
1. Clone the repository:
   ```bash
   git clone https://github.com/mims1234/polygon-survival-game.git
   ```
2. Navigate into the project directory:
   ```bash
   cd polygon-survival-game
   ```
3. Boot up a local web server (If you are using VS Code, you can use the *Live Server* extension to run the `index.html` file directly).

## Controls
- **1-5 Keys**: Quick-select tools (Hand, Axe, Pickaxe, Bucket, Build Mode)
- **M**: Toggle the Map panel
- **H**: View Help/Instructions Menu
- **Left-Click (Grid)**: Walk to tiles and interact/queue resource gathering or building.
